## hma vpn
~~hide my ass~~

**无广告，安卓电脑都可以用**

*我最早用的一个windowvpn，感觉还不错，就是后面几天网速一般*

**白嫖方法：**
> 首先要有个国外的idcard，nemso-gen.com可以生成，输入某个地区的bin比如453590，
